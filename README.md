# B站直播弹幕过滤 & 一键拉黑 (BLS)

[![Tampermonkey](https://img.shields.io/badge/Tampermonkey-✔-green.svg)](https://www.tampermonkey.net/)
[![Version](https://img.shields.io/badge/version-2.0.8-blue.svg)]()

> 专为 B 站直播设计的弹幕过滤脚本，支持按 UID、粉丝牌、荣耀等级屏蔽弹幕，并提供一键拉黑功能。  
> **轻量 · 跨帧通信 · 配置即时生效**
>
> 
> **仅支持聊天栏23333,视频弹幕手动关闭吧**
>
> 
> **由MINIMAX,DEEPSEEK编写**
---

## ✨ 功能亮点

- **🔹 UID 拉黑** – 手动添加 UID，拉黑后该用户所有弹幕（聊天框 + 视频滚动）都会被屏蔽。
- **🔹 粉丝牌过滤** – 按粉丝牌名称 + 等级上限屏蔽弹幕（例如：屏蔽“三千八”等级 ≤ 10 的弹幕）。
- **🔹 荣耀等级过滤** – 通过图标映射表，屏蔽指定等级及以下的荣耀勋章弹幕。
- **🔹 屏蔽无牌/无荣耀用户** – 一键过滤没有佩戴粉丝牌或荣耀勋章的用户。
- **🔹 智能图标扫描** – 从聊天框自动抓取所有荣耀勋章图标，用户只需填写对应等级即可建立映射。
- **🔹 一键保存全部映射** – 扫描结果支持批量填写并一键保存，告别逐条保存的繁琐。
- **🔹 配置即时生效** – 所有修改无需刷新页面，自动同步到所有帧并刷新弹幕显示。
- **🔹 显示被屏蔽弹幕** – 可选择将屏蔽的弹幕变灰保留，方便调试。
- **🔹 导出/导入配置** – 支持 JSON 格式导出和导入，便于备份或跨设备同步。

## 📦 安装方法

1. **安装油猴扩展**（Tampermonkey）  
   - Chrome / Edge: [Tampermonkey 商店链接](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)  
   - Firefox: [Tampermonkey 附加组件](https://addons.mozilla.org/firefox/addon/tampermonkey/)

2. **安装本脚本**   
   - 复制完整脚本内容，在 Tampermonkey 中新建脚本并粘贴保存。

3. **打开 B 站任意直播间**，脚本自动运行，右下角会出现粉色圆形浮动按钮（`BLS`）。

4. **导入荣耀等级映射脚本**:bls-config-1781792577813

## 🎮 使用方法

### 打开配置面板
点击右下角的 **`BLS`** 按钮，即可展开配置面板。再次点击则收起。

### 面板功能介绍

#### ① UID 拉黑
- 在输入框中粘贴或输入 UID，点击 **「加入」**，该用户弹幕会被屏蔽。
- 拉黑的 UID 会显示在下方列表中，点击 **「移除」** 可取消拉黑。

#### ② 粉丝牌过滤
- 输入粉丝牌名称（例如 `三千八`）和等级上限（例如 `10`），点击 **「加入」**。
- 此后所有该粉丝牌且等级 ≤ 10 的弹幕将被屏蔽。
- 可添加多条规则，支持不同粉丝牌。

#### ③ 荣耀等级过滤
- **屏蔽等级阈值**：设置一个数值（例如 `20`），所有荣耀等级 ≤ 20 的弹幕将被屏蔽。设为 `0` 表示关闭此功能。
- **图标映射表**：由于 B 站前端不直接显示等级，脚本通过图标 URL 来识别荣耀等级。您需要建立「图标 URL → 等级数字」的映射。
  - **自动扫描**：点击 **「从聊天框扫描图标」**，脚本会自动扫描当前聊天框中的所有荣耀勋章图标，并列出尚未映射的新图标，您只需为每个图标填入对应的等级数字，然后点击 **「💾 一键保存所有」** 即可。
  - **手动添加**：如果您已知图标 URL 和对应等级，可直接在下方输入框填写，点击 **「加入映射」**。
- 已建立的映射会按等级 **升序** 显示在列表中，方便查看。

#### ④ 通用开关
- **显示被屏蔽的弹幕（灰色）**：开启后，被屏蔽的弹幕不会消失，而是变为灰色半透明，方便调试。
- **屏蔽无粉丝牌的用户**：一键屏蔽所有未佩戴粉丝牌的用户弹幕。
- **屏蔽无荣耀等级的用户**：一键屏蔽所有未佩戴荣耀勋章的用户弹幕。

#### ⑤ 配置导入/导出
- **导出配置 JSON**：将当前所有设置（UID 黑名单、粉丝牌规则、图标映射等）导出为 `.json` 文件。
- **导入配置 JSON**：选择之前导出的文件，覆盖当前配置（需要确认）。

## 🔧 高级说明

### 跨帧通信原理
B 站直播页面由多个 iframe 组成（聊天框、播放器等均独立）。本脚本利用 `GM_setValue` / `GM_addValueChangeListener` 实现顶层窗口与 iframe 之间的数据同步和扫描请求响应，确保配置更改即时生效，且扫描功能能够正确获取聊天框内的图标。

### 数据存储
所有配置保存在浏览器的 `GM_setValue` 中，键名为 `bls_config_v2`。您可以通过导出/导入功能进行备份。

### 注意事项
- 拉黑功能需要登录 B 站，且依赖 `bili_jct` Cookie（脚本自动获取）。
- 荣耀等级映射依赖于图标 URL，若 B 站更换图标地址，需重新扫描。
- 视频画面上的滚动弹幕（Canvas/WebGL 渲染）无法通过 DOM 操作屏蔽，但通过 **拉黑 UID** 后，B 站会全局屏蔽该用户的所有弹幕（包括滚动弹幕）。
